Compiler Series

c++, ocaml, ruby, compiler-construction

A set of three projects for Programming Language Design & Implementation. These projects outline the core components and functionality of a compiler for a simple programming language:

  • The first is a scanner/parser/interpreter written in C++ that implements exception-based error handling.
  • The second is a scanner/parser/interpreter written in OCaml that generates equivalent, fully executable C-code after interpreting a program.
  • The final is a cross-indexer written in Ruby which creates a series of html pages linking identifiers to declarations. It does this by using the DWARF symbol-table output generated by gcc. No semantic analysis is done, but the accuracy is still quite good, and the entire program (html generation and all) is less than 200 lines.
(In a private GitHub repository. Code can be shared upon request.)

Java Street Mapper

java, swing, gui

Click on title to download the Jar file for this project.

Project for The Science of Data Structures that required us to create a GUI in Java to find the shortest path (using Dijkstra's algorithm) between two points on the real road network in Monroe County, New York. Also calculates the minimum spanning tree for the network.

Click two points on the grid to find the shortest road distance between them, or click on one point and drag to find shortest distance between your starting point and your mouse as a live update. Click again to choose a new starting point. Right click to clear points.

Coordinates for your current location are shown in the bottom left, and distance between points in the bottom right. (Not quite sure what the units are, but the distance seems approximately accurate interpreted as hectometers... how odd)
